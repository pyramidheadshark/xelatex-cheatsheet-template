3. Обратное Распространение Ошибки (Backpropagation)

Цель: Вычислить градиенты функции потерь JJJ по всем параметрам сети (W[l]W^{[l]}W[l] и b[l]b^{[l]}b[l] для всех слоев lll). Мы используем цепное правило, двигаясь от последнего слоя к первому.

δj[l]=∂J∂zj[l]\delta_j^{[l]} = \frac{\partial J}{\partial z_j^{[l]}} δj[l]​=∂zj[l]​∂J​

    • Матричная форма: Собирая все производные ∂J∂Wjk[l]\frac{\partial J}{\partial W_{jk}^{[l]}}∂Wjk[l]​∂J​ в матрицу ∂J∂W[l]\frac{\partial J}{\partial W^{[l]}}∂W[l]∂J​ того же размера, что и W[l]W^{[l]}W[l], получаем:

      ∂J∂W[l]=δ[l](a[l−1])T \boxed{\frac{\partial J}{\partial W^{[l]}} = \delta^{[l]} (a^{[l-1]})^T} ∂W[l]∂J​=δ[l](a[l−1])T​

      • Объяснение: Градиент по матрице весов слоя lll равен внешнему произведению вектора ошибки этого слоя δ[l]\delta^{[l]}δ[l] (размер n[l]×1n^{[l]} \times 1n[l]×1) на транспонированный вектор активаций предыдущего слоя (a[l−1])T(a^{[l-1]})^T(a[l−1])T (размер 1×n[l−1]1 \times n^{[l-1]}1×n[l−1]). Результат имеет размер n[l]×n[l−1]n^{[l]} \times n^{[l-1]}n[l]×n[l−1], как и W[l]W^{[l]}W[l]. (Примечание: При усреднении по батчу размером BBB здесь будет 1B∑i∈Batchδ[l](i)(a[l−1](i))T\frac{1}{B} \sum_{i \in Batch} \delta^{[l](i)} (a^{[l-1](i)})^TB1​∑i∈Batch​δ[l](i)(a[l−1](i))T).

  • Градиент по Смещениям b[l]b^{[l]}b[l]:

    ∂J∂bj[l]=∂J∂zj[l]⋅∂zj[l]∂bj[l]\frac{\partial J}{\partial b_j^{[l]}} = \frac{\partial J}{\partial z_j^{[l]}} \cdot \frac{\partial z_j^{[l]}}{\partial b_j^{[l]}} ∂bj[l]​∂J​=∂zj[l]​∂J​⋅∂bj[l]​∂zj[l]​​

    Первый множитель - δj[l]\delta_j^{[l]}δj[l]​. Второй множитель:

    ∂zj[l]∂bj[l]=∂∂bj[l](∑p=1n[l−1]Wjp[l]ap[l−1]+bj[l])=1\frac{\partial z_j^{[l]}}{\partial b_j^{[l]}} = \frac{\partial}{\partial b_j^{[l]}} \left( \sum_{p=1}^{n^{[l-1]}} W_{jp}^{[l]} a_p^{[l-1]} + b_j^{[l]} \right) = 1 ∂bj[l]​∂zj[l]​​=∂bj[l]​∂​​p=1∑n[l−1]​Wjp[l]​ap[l−1]​+bj[l]​​=1

    Получаем:

    ∂J∂bj[l]=δj[l]\frac{\partial J}{\partial b_j^{[l]}} = \delta_j^{[l]} ∂bj[l]​∂J​=δj[l]​

    • Векторная форма:

      ∂J∂b[l]=δ[l] \boxed{\frac{\partial J}{\partial b^{[l]}} = \delta^{[l]}} ∂b[l]∂J​=δ[l]​

      • Объяснение: Градиент по вектору смещений слоя lll просто равен вектору ошибки этого слоя δ[l]\delta^{[l]}δ[l]. (Примечание: При усреднении по батчу будет 1B∑i∈Batchδ[l](i)\frac{1}{B} \sum_{i \in Batch} \delta^{[l](i)}B1​∑i∈Batch​δ[l](i)).

Итог Backpropagation: Мы получили формулы для вычисления ошибки δ[l]\delta^{[l]}δ[l] для каждого слоя (двигаясь назад от LLL до 1) и, используя эти ошибки, вычислили градиенты по параметрам ∂J∂W[l]\frac{\partial J}{\partial W^{[l]}}∂W[l]∂J​ и ∂J∂b[l]\frac{\partial J}{\partial b^{[l]}}∂b[l]∂J​ для каждого слоя.

4. Обновление Весов (Градиентный Спуск)

Используя вычисленные градиенты, обновляем параметры сети:

Скалярная форма (для каждого веса и смещения):

Wjk[l]:=Wjk[l]−α∂J∂Wjk[l]bj[l]:=bj[l]−α∂J∂bj[l]\boxed{ \begin{aligned} W_{jk}^{[l]} &:= W_{jk}^{[l]} - \alpha \frac{\partial J}{\partial W_{jk}^{[l]}} \\ b_j^{[l]} &:= b_j^{[l]} - \alpha \frac{\partial J}{\partial b_j^{[l]}} \end{aligned} } Wjk[l]​bj[l]​​:=Wjk[l]​−α∂Wjk[l]​∂J​:=bj[l]​−α∂bj[l]​∂J​​​

Векторная/Матричная форма (для каждого слоя lll):

W[l]:=W[l]−α∂J∂W[l]b[l]:=b[l]−α∂J∂b[l]\boxed{ \begin{aligned} W^{[l]} &:= W^{[l]} - \alpha \frac{\partial J}{\partial W^{[l]}} \\ b^{[l]} &:= b^{[l]} - \alpha \frac{\partial J}{\partial b^{[l]}} \end{aligned} } W[l]b[l]​:=W[l]−α∂W[l]∂J​:=b[l]−α∂b[l]∂J​​​

  • Объяснение: Параметры обновляются шагом в направлении анти-градиента. Вместо простого GD могут использоваться более сложные оптимизаторы (Adam и т.д.), которые модифицируют это правило обновления.

5. Batch Normalization (Формулы Прямого Прохода)

(Для одного признака (канала) kkk на входе слоя zkz_kzk​, по мини-батчу BBB)

Шаг 1: Среднее по батчу μB,k\mu_{B,k}μB,k​

μB,k=1B∑i=1Bzk(i)\mu_{B,k} = \frac{1}{B} \sum_{i=1}^{B} z_k^{(i)} μB,k​=B1​i=1∑B​zk(i)​

Шаг 2: Дисперсия по батчу σB,k2\sigma^2_{B,k}σB,k2​

σB,k2=1B∑i=1B(zk(i)−μB,k)2\sigma^2_{B,k} = \frac{1}{B} \sum_{i=1}^{B} (z_k^{(i)} - \mu_{B,k})^2 σB,k2​=B1​i=1∑B​(zk(i)​−μB,k​)2

Шаг 3: Нормализация z^k(i)\hat{z}_k^{(i)}z^k(i)​

z^k(i)=zk(i)−μB,kσB,k2+ϵ\hat{z}_k^{(i)} = \frac{z_k^{(i)} - \mu_{B,k}}{\sqrt{\sigma^2_{B,k} + \epsilon}} z^k(i)​=σB,k2​+ϵ​zk(i)​−μB,k​​

  • ϵ\epsilonϵ: Малая константа для численной стабильности (например, 10−810^{-8}10−8).

Шаг 4: Масштабирование и Сдвиг BN(zk(i))BN(z_k^{(i)})BN(zk(i)​)

BN(zk(i))=γkz^k(i)+βk\boxed{ BN(z_k^{(i)}) = \gamma_k \hat{z}_k^{(i)} + \beta_k } BN(zk(i)​)=γk​z^k(i)​+βk​​

  • γk,βk\gamma_k, \beta_kγk​,βk​: Обучаемые параметры слоя Batch Normalization для признака kkk.

(Вывод градиентов для Batch Normalization сложнее и выходит за рамки базового обзора, но важно знать, что γk\gamma_kγk​ и βk\beta_kβk​ обновляются через backpropagation, как и другие параметры сети).
