1. Общая Идея Регуляризации

Цель - предотвратить переобучение, добавив штраф за сложность модели (большие веса) к исходной функции потерь.

Общая Формула Регуляризованной Функции Потерь:

Jreg(w)=Joriginal(w)+Penalty(w)J_{reg}(w) = J_{original}(w) + \text{Penalty}(w) Jreg​(w)=Joriginal​(w)+Penalty(w)

  • Объяснение: Мы минимизируем не только ошибку на обучающих данных (JoriginalJ_{original}Joriginal​), но и "штраф" за величину весов (Penalty\text{Penalty}Penalty). Это заставляет модель искать баланс между точностью на трейне и простотой (меньшими весами).

2. L2 Регуляризация (Гребневая / Ridge)

Штрафует модель пропорционально сумме квадратов весов (кроме w0w_0w0​).

Определение Штрафа L2 (Penalty):

PenaltyL2(w)=λ2m∑j=1nwj2\text{Penalty}_{L2}(w) = \frac{\lambda}{2m} \sum_{j=1}^{n} w_j^2 PenaltyL2​(w)=2mλ​j=1∑n​wj2​

  • λ≥0\lambda \ge 0λ≥0: Гиперпараметр, контролирующий силу регуляризации.
  • ∑j=1nwj2\sum_{j=1}^{n} w_j^2∑j=1n​wj2​: Сумма квадратов весов только для признаков (от w1w_1w1​ до wnw_nwn​). w0w_0w0​ не штрафуется.
  • 12m\frac{1}{2m}2m1​: Коэффициент для удобства и согласования с JoriginalJ_{original}Joriginal​ (часто берут Joriginal=12m∑(… )2J_{original} = \frac{1}{2m}\sum(\dots)^2Joriginal​=2m1​∑(…)2). Ключевой элемент - λ∑wj2\lambda \sum w_j^2λ∑wj2​.

Регуляризованная Функция Потерь JRidge(w)J_{Ridge}(w)JRidge​(w) (Пример с MSE):

JRidge(w)=12m∑i=1m(y^(i)−y(i))2⏟Joriginal (MSE)+λ2m∑j=1nwj2⏟L2 PenaltyJ_{Ridge}(w) = \underbrace{\frac{1}{2m} \sum_{i=1}^{m} (\hat{y}^{(i)} - y^{(i)})^2}_{J_{original} \text{ (MSE)}} + \underbrace{\frac{\lambda}{2m} \sum_{j=1}^{n} w_j^2}_{\text{L2 Penalty}} JRidge​(w)=Joriginal​ (MSE)2m1​i=1∑m​(y^​(i)−y(i))2​​+L2 Penalty2mλ​j=1∑n​wj2​​​

  • Объяснение: Минимизируя JRidgeJ_{Ridge}JRidge​, мы одновременно минимизируем ошибку MSE и сумму квадратов весов.

Градиент L2 Регуляризации: Нам нужно найти производные JRidge(w)J_{Ridge}(w)JRidge​(w) по каждому весу wkw_kwk​. Производная суммы равна сумме производных: ∂JRidge∂wk=∂Joriginal∂wk+∂(PenaltyL2)∂wk\frac{\partial J_{Ridge}}{\partial w_k} = \frac{\partial J_{original}}{\partial w_k} + \frac{\partial (\text{Penalty}_{L2})}{\partial w_k}∂wk​∂JRidge​​=∂wk​∂Joriginal​​+∂wk​∂(PenaltyL2​)​.

Шаг 1: Производная Штрафа L2 ∂(PenaltyL2)∂wk\frac{\partial (\text{Penalty}_{L2})}{\partial w_k}∂wk​∂(PenaltyL2​)​

  • Случай 1: k=0k = 0k=0 (для свободного члена w0w_0w0​) Штраф PenaltyL2\text{Penalty}_{L2}PenaltyL2​ не зависит от w0w_0w0​ (суммирование с j=1j=1j=1).

    ∂(PenaltyL2)∂w0=0\frac{\partial (\text{Penalty}_{L2})}{\partial w_0} = 0 ∂w0​∂(PenaltyL2​)​=0

  • Случай 2: k≥1k \ge 1k≥1 (для веса признака wkw_kwk​)

    ∂(PenaltyL2)∂wk=∂∂wk(λ2m∑j=1nwj2)\frac{\partial (\text{Penalty}_{L2})}{\partial w_k} = \frac{\partial}{\partial w_k} \left( \frac{\lambda}{2m} \sum_{j=1}^{n} w_j^2 \right) ∂wk​∂(PenaltyL2​)​=∂wk​∂​(2mλ​j=1∑n​wj2​)

    Выносим константу и сумму (производная суммы равна сумме производных):

    =λ2m∑j=1n∂∂wk(wj2)= \frac{\lambda}{2m} \sum_{j=1}^{n} \frac{\partial}{\partial w_k} (w_j^2) =2mλ​j=1∑n​∂wk​∂​(wj2​)

    Производная ∂∂wk(wj2)\frac{\partial}{\partial w_k} (w_j^2)∂wk​∂​(wj2​) равна 0, если j≠kj \ne kj=k, и равна 2wk2w_k2wk​, если j=kj = kj=k. Поэтому из всей суммы остается только одно слагаемое при j=kj=kj=k:

    =λ2m(2wk)=λmwk= \frac{\lambda}{2m} (2 w_k) = \frac{\lambda}{m} w_k =2mλ​(2wk​)=mλ​wk​

    • Объяснение: Производная L2 штрафа по весу wkw_kwk​ пропорциональна самому весу wkw_kwk​.

Шаг 2: Полный Градиент JRidge(w)J_{Ridge}(w)JRidge​(w)

  • Для w0w_0w0​ (k=0k=0k=0):

    ∂JRidge∂w0=∂Joriginal∂w0+0=∂Joriginal∂w0\boxed{ \frac{\partial J_{Ridge}}{\partial w_0} = \frac{\partial J_{original}}{\partial w_0} + 0 = \frac{\partial J_{original}}{\partial w_0} } ∂w0​∂JRidge​​=∂w0​∂Joriginal​​+0=∂w0​∂Joriginal​​​

    (Производная по w0w_0w0​ такая же, как и без регуляризации).
  • Для wkw_kwk​ (k=1,...,nk=1, ..., nk=1,...,n):

    ∂JRidge∂wk=∂Joriginal∂wk+λmwk\boxed{ \frac{\partial J_{Ridge}}{\partial w_k} = \frac{\partial J_{original}}{\partial w_k} + \frac{\lambda}{m} w_k } ∂wk​∂JRidge​​=∂wk​∂Joriginal​​+mλ​wk​​

    (К исходному градиенту добавляется член λmwk\frac{\lambda}{m} w_kmλ​wk​).

Обновление Весов в Градиентном Спуске с L2:

  • Для w0w_0w0​ (k=0k=0k=0):

    w0:=w0−α∂Joriginal∂w0w_0 := w_0 - \alpha \frac{\partial J_{original}}{\partial w_0} w0​:=w0​−α∂w0​∂Joriginal​​

  • Для wkw_kwk​ (k=1,...,nk=1, ..., nk=1,...,n):

    wk:=wk−α(∂Joriginal∂wk+λmwk)w_k := w_k - \alpha \left( \frac{\partial J_{original}}{\partial w_k} + \frac{\lambda}{m} w_k \right) wk​:=wk​−α(∂wk​∂Joriginal​​+mλ​wk​)

    Перегруппируем члены:

    wk:=wk−αλmwk−α∂Joriginal∂wkw_k := w_k - \alpha \frac{\lambda}{m} w_k - \alpha \frac{\partial J_{original}}{\partial w_k} wk​:=wk​−αmλ​wk​−α∂wk​∂Joriginal​​

    wk:=wk(1−αλm)−α∂Joriginal∂wk\boxed{ w_k := w_k \left(1 - \alpha \frac{\lambda}{m}\right) - \alpha \frac{\partial J_{original}}{\partial w_k} } wk​:=wk​(1−αmλ​)−α∂wk​∂Joriginal​​​

    • Объяснение (Weight Decay): Формула показывает, что перед выполнением основного шага градиентного спуска (вычитание α∂Joriginal∂wk\alpha \frac{\partial J_{original}}{\partial w_k}α∂wk​∂Joriginal​​), вес wkw_kwk​ умножается на коэффициент (1−αλm)(1 - \alpha \frac{\lambda}{m})(1−αmλ​). Так как α,λ,m\alpha, \lambda, mα,λ,m положительны, этот коэффициент меньше 1 (при разумной α\alphaα). Это означает, что вес wkw_kwk​ на каждой итерации немного "уменьшается" или "затухает" (weight decay) по направлению к нулю. Величина затухания пропорциональна текущему значению веса wkw_kwk​. Это и есть механизм "сжатия" весов в L2 регуляризации.

3. L1 Регуляризация (Lasso)

Штрафует модель пропорционально сумме модулей весов (кроме w0w_0w0​).

Определение Штрафа L1 (Penalty):

PenaltyL1(w)=λm∑j=1n∣wj∣\text{Penalty}_{L1}(w) = \frac{\lambda}{m} \sum_{j=1}^{n} |w_j| PenaltyL1​(w)=mλ​j=1∑n​∣wj​∣

  • ∣wj∣|w_j|∣wj​∣: Модуль (абсолютное значение) веса wjw_jwj​.
  • Остальные обозначения (λ,m,∑j=1n\lambda, m, \sum_{j=1}^{n}λ,m,∑j=1n​) те же, что и для L2.

Регуляризованная Функция Потерь JLasso(w)J_{Lasso}(w)JLasso​(w) (Пример с MSE):

JLasso(w)=12m∑i=1m(y^(i)−y(i))2⏟Joriginal (MSE)+λm∑j=1n∣wj∣⏟L1 PenaltyJ_{Lasso}(w) = \underbrace{\frac{1}{2m} \sum_{i=1}^{m} (\hat{y}^{(i)} - y^{(i)})^2}_{J_{original} \text{ (MSE)}} + \underbrace{\frac{\lambda}{m} \sum_{j=1}^{n} |w_j|}_{\text{L1 Penalty}} JLasso​(w)=Joriginal​ (MSE)2m1​i=1∑m​(y^​(i)−y(i))2​​+L1 Penaltymλ​j=1∑n​∣wj​∣​​

  • Объяснение: Минимизируя JLassoJ_{Lasso}JLasso​, мы одновременно минимизируем ошибку MSE и сумму модулей весов.

Градиент L1 Регуляризации (Проблема и Решение):

Проблема: Функция модуля ∣wk∣|w_k|∣wk​∣ не дифференцируема в точке wk=0w_k = 0wk​=0. У нее нет однозначно определенной производной в этой точке.

Решение: Использование Субградиента Субградиент - это обобщение понятия градиента для негладких выпуклых функций. Для модуля ∣wk∣|w_k|∣wk​∣ субградиент ∂∣wk∣\partial |w_k|∂∣wk​∣ определяется как:

∂∣wk∣={+1,if wk>0−1,if wk<0любое число в [−1,1],if wk=0\partial |w_k| = \begin{cases} +1, & \text{if } w_k > 0 \\ -1, & \text{if } w_k < 0 \\ \text{любое число в } [-1, 1], & \text{if } w_k = 0 \end{cases} ∂∣wk​∣=⎩⎨⎧​+1,−1,любое число в [−1,1],​if wk​>0if wk​<0if wk​=0​

Часто используют обозначение sign(wk)\text{sign}(w_k)sign(wk​) для субградиента, где sign(0)\text{sign}(0)sign(0) может быть принято равным 0 (хотя это не единственный вариант) для простоты реализации алгоритмов.

sign(wk)={+1,if wk>0−1,if wk<00,if wk=0\text{sign}(w_k) = \begin{cases} +1, & \text{if } w_k > 0 \\ -1, & \text{if } w_k < 0 \\ 0, & \text{if } w_k = 0 \end{cases} sign(wk​)=⎩⎨⎧​+1,−1,0,​if wk​>0if wk​<0if wk​=0​

Шаг 1: Субградиент Штрафа L1 ∂(PenaltyL1)/∂wk\partial (\text{Penalty}_{L1}) / \partial w_k∂(PenaltyL1​)/∂wk​

  • Случай 1: k=0k = 0k=0 (для свободного члена w0w_0w0​)

    ∂(PenaltyL1)∂w0=0\frac{\partial (\text{Penalty}_{L1})}{\partial w_0} = 0 ∂w0​∂(PenaltyL1​)​=0

  • Случай 2: k≥1k \ge 1k≥1 (для веса признака wkw_kwk​) Используя субградиент для ∣wj∣|w_j|∣wj​∣:

    ∂(PenaltyL1)∂wk=∂∂wk(λm∑j=1n∣wj∣)=λm∑j=1n∂∣wj∣∂wk\frac{\partial (\text{Penalty}_{L1})}{\partial w_k} = \frac{\partial}{\partial w_k} \left( \frac{\lambda}{m} \sum_{j=1}^{n} |w_j| \right) = \frac{\lambda}{m} \sum_{j=1}^{n} \frac{\partial |w_j|}{\partial w_k} ∂wk​∂(PenaltyL1​)​=∂wk​∂​(mλ​j=1∑n​∣wj​∣)=mλ​j=1∑n​∂wk​∂∣wj​∣​

    Производная (субградиент) ∂∣wj∣∂wk\frac{\partial |w_j|}{\partial w_k}∂wk​∂∣wj​∣​ равна 0, если j≠kj \ne kj=k, и равна sign(wk)\text{sign}(w_k)sign(wk​), если j=kj = kj=k.

    ∂(PenaltyL1)∂wk=λmsign(wk)\frac{\partial (\text{Penalty}_{L1})}{\partial w_k} = \frac{\lambda}{m} \text{sign}(w_k) ∂wk​∂(PenaltyL1​)​=mλ​sign(wk​)

    • Объяснение: Субградиент L1 штрафа по весу wkw_kwk​ равен константе (λ/m\lambda/mλ/m или −λ/m-\lambda/m−λ/m), знак которой зависит от знака wkw_kwk​.

Шаг 2: Полный Субградиент JLasso(w)J_{Lasso}(w)JLasso​(w)

  • Для w0w_0w0​ (k=0k=0k=0):

    ∂JLasso∂w0=∂Joriginal∂w0\boxed{ \frac{\partial J_{Lasso}}{\partial w_0} = \frac{\partial J_{original}}{\partial w_0} } ∂w0​∂JLasso​​=∂w0​∂Joriginal​​​

  • Для wkw_kwk​ (k=1,...,nk=1, ..., nk=1,...,n):

    ∂JLasso∂wk=∂Joriginal∂wk+λmsign(wk)\boxed{ \frac{\partial J_{Lasso}}{\partial w_k} = \frac{\partial J_{original}}{\partial w_k} + \frac{\lambda}{m} \text{sign}(w_k) } ∂wk​∂JLasso​​=∂wk​∂Joriginal​​+mλ​sign(wk​)​

Обновление Весов в Субградиентном Спуске с L1 (Концептуально):

  • Для w0w_0w0​ (k=0k=0k=0):

    w0:=w0−α∂Joriginal∂w0w_0 := w_0 - \alpha \frac{\partial J_{original}}{\partial w_0} w0​:=w0​−α∂w0​∂Joriginal​​

  • Для wkw_kwk​ (k=1,...,nk=1, ..., nk=1,...,n):

    wk:=wk−α(∂Joriginal∂wk+λmsign(wk))\boxed{ w_k := w_k - \alpha \left( \frac{\partial J_{original}}{\partial w_k} + \frac{\lambda}{m} \text{sign}(w_k) \right) } wk​:=wk​−α(∂wk​∂Joriginal​​+mλ​sign(wk​))​

  • Объяснение (Почему L1 обнуляет веса): Рассмотрим член регуляризации −αλmsign(wk)-\alpha \frac{\lambda}{m} \text{sign}(w_k)−αmλ​sign(wk​). Он вычитает константу (αλ/m\alpha \lambda / mαλ/m), если wk>0w_k > 0wk​>0, и прибавляет константу, если wk<0w_k < 0wk​<0. Это означает, что L1 регуляризация "тянет" вес к нулю с постоянной силой, независимо от величины веса (в отличие от L2, где сила λmwk\frac{\lambda}{m} w_kmλ​wk​ уменьшается по мере приближения к нулю). Из-за этой постоянной "тяги" веса, которые становятся достаточно малыми, могут пересечь ноль и стать равными нулю за конечное число шагов. Это и есть механизм отбора признаков в Lasso. (Примечание: На практике для L1 часто используют более сложные Проксимальные или Координатные методы спуска, которые корректнее обрабатывают точку wk=0w_k=0wk​=0).

4. Elastic Net

Комбинирует L1 и L2 регуляризацию.

Определение Штрафа Elastic Net:

PenaltyElasticNet(w)=λ1m∑j=1n∣wj∣⏟L1 часть+λ22m∑j=1nwj2⏟L2 часть\text{Penalty}_{ElasticNet}(w) = \underbrace{\frac{\lambda_1}{m} \sum_{j=1}^{n} |w_j|}_{\text{L1 часть}} + \underbrace{\frac{\lambda_2}{2m} \sum_{j=1}^{n} w_j^2}_{\text{L2 часть}} PenaltyElasticNet​(w)=L1 частьmλ1​​j=1∑n​∣wj​∣​​+L2 часть2mλ2​​j=1∑n​wj2​​​

  • Или с параметром смешивания αmix∈[0,1]\alpha_{mix} \in [0, 1]αmix​∈[0,1]:

PenaltyElasticNet(w)=λ(αmix∑j=1n∣wj∣+(1−αmix)12∑j=1nwj2)\text{Penalty}_{ElasticNet}(w) = \lambda \left( \alpha_{mix} \sum_{j=1}^{n} |w_j| + (1 - \alpha_{mix}) \frac{1}{2} \sum_{j=1}^{n} w_j^2 \right) PenaltyElasticNet​(w)=λ(αmix​j=1∑n​∣wj​∣+(1−αmix​)21​j=1∑n​wj2​)

  • Объяснение: Позволяет одновременно использовать преимущества L1 (разреженность, отбор признаков) и L2 (стабильность при коррелирующих признаках, сжатие). Требует подбора двух гиперпараметров (λ1,λ2\lambda_1, \lambda_2λ1​,λ2​ или λ,αmix\lambda, \alpha_{mix}λ,αmix​) и сложных методов оптимизации.

5. Важность Масштабирования Признаков

  • Необходимость: Перед применением ЛЮБОЙ регуляризации (L1, L2, Elastic Net) крайне важно масштабировать признаки x1,...,xnx_1, ..., x_nx1​,...,xn​, чтобы они имели примерно одинаковый диапазон значений.
  • Причина: Штрафы λ∑wj2\lambda \sum w_j^2λ∑wj2​ и λ∑∣wj∣\lambda \sum |w_j|λ∑∣wj​∣ зависят от величины весов wjw_jwj​. Если признаки не масштабированы, то веса wjw_jwj​ будут отражать не только важность признака, но и его масштаб. Регуляризация будет непропорционально сильно штрафовать веса признаков с большим исходным масштабом, что некорректно. Масштабирование уравнивает "стартовые условия" для всех признаков.
  • Методы: StandardScaler, MinMaxScaler (применять fit_transform на трейне, transform на валидации/тесте).
