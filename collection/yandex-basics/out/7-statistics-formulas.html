Детальный Вывод Формул и Концепций: Статистическая Значимость

(Используем обозначения: mmm - число объектов, nnn - число признаков, KKK - число фолдов в CV, NNN - размер тестовой выборки, MA,MBM_A, M_BMA​,MB​ - значения метрики для моделей A и B, Mˉ\bar{M}Mˉ - среднее значение метрики, sMs_MsM​ - стандартное отклонение метрики, dk=MA,k−MB,kd_k = M_{A,k} - M_{B,k}dk​=MA,k​−MB,k​ - разность метрик на kkk-ом фолде, dˉ\bar{d}dˉ - средняя разность, sds_dsd​ - стандартное отклонение разностей, α\alphaα - уровень значимости, μM,μd\mu_M, \mu_dμM​,μd​ - истинные средние значения, CI - доверительный интервал, SE - стандартная ошибка, RRR - число бутстрап-итераций).

1. Оценка Метрики и ее Вариативность

  • Проблема: Метрика MMM, посчитанная на конечной выборке (тестовый сет или фолд CV) размера NNN, является оценкой истинного значения μM\mu_MμM​. Эта оценка имеет дисперсию (variance) из-за случайности выборки.
  • Стандартная Ошибка Среднего (Standard Error of the Mean - SE): Если у нас есть несколько оценок метрики (например, M1,...,MKM_1, ..., M_KM1​,...,MK​ по фолдам CV), мы можем оценить стандартное отклонение среднего значения Mˉ\bar{M}Mˉ. Это и есть стандартная ошибка.

    SEMˉ=sMKSE_{\bar{M}} = \frac{s_M}{\sqrt{K}} SEMˉ​=K​sM​​

    • sMs_MsM​: Выборочное стандартное отклонение значений метрики M1,...,MKM_1, ..., M_KM1​,...,MK​.

      sM=1K−1∑k=1K(Mk−Mˉ)2 s_M = \sqrt{\frac{1}{K-1} \sum_{k=1}^{K} (M_k - \bar{M})^2} sM​=K−11​k=1∑K​(Mk​−Mˉ)2​

    • KKK: Количество фолдов (или независимых оценок).
    • Объяснение: SE показывает, насколько в среднем наше выборочное среднее Mˉ\bar{M}Mˉ отклоняется от истинного среднего μM\mu_MμM​. Чем больше KKK, тем меньше SE, тем надежнее наша оценка Mˉ\bar{M}Mˉ.

2. Доверительный Интервал (Confidence Interval - CI) для Средней Метрики

  • Цель: Оценить диапазон, в котором с заданной уверенностью (обычно 95%) лежит истинное среднее значение метрики μM\mu_MμM​.
  • Формула (на основе результатов CV):

    CI for μM=Mˉ±Погрешность\text{CI for } \mu_M = \bar{M} \pm \text{Погрешность} CI for μM​=Mˉ±Погрешность

    Погрешность=tα/2,K−1⋅SEMˉ=tα/2,K−1⋅sMK\text{Погрешность} = t_{\alpha/2, K-1} \cdot SE_{\bar{M}} = t_{\alpha/2, K-1} \cdot \frac{s_M}{\sqrt{K}} Погрешность=tα/2,K−1​⋅SEMˉ​=tα/2,K−1​⋅K​sM​​

    • Mˉ\bar{M}Mˉ: Среднее значение метрики по KKK фолдам.
    • sMs_MsM​: Стандартное отклонение метрики по KKK фолдам.
    • KKK: Количество фолдов.
    • tα/2,K−1t_{\alpha/2, K-1}tα/2,K−1​: Критическое значение (квантиль) t-распределения Стьюдента с K−1K-1K−1 степенями свободы для уровня доверия 1−α1-\alpha1−α.
      • Для 95% CI, α=0.05\alpha = 0.05α=0.05, и мы ищем t0.025,K−1t_{0.025, K-1}t0.025,K−1​.
      • При большом KKK (например, K≥30K \ge 30K≥30), t-распределение близко к нормальному, и можно использовать z-квантиль zα/2≈1.96z_{\alpha/2} \approx 1.96zα/2​≈1.96 для 95% CI.
    • Объяснение: Мы берем нашу точечную оценку Mˉ\bar{M}Mˉ и добавляем/вычитаем "запас" (погрешность), который зависит от разброса оценок (sMs_MsM​), количества оценок (KKK) и нашего желаемого уровня уверенности (через ttt-квантиль). Чем больше разброс или меньше фолдов, тем шире будет интервал.

3. Сравнение Моделей A и B с Помощью CV: Парный t-тест

  • Цель: Проверить гипотезу о том, что средние значения метрик моделей A и B равны, используя результаты CV.
  • Данные: Пары метрик (MA,1,MB,1),...,(MA,K,MB,K)(M_{A,1}, M_{B,1}), ..., (M_{A,K}, M_{B,K})(MA,1​,MB,1​),...,(MA,K​,MB,K​). Рассчитываем разности dk=MA,k−MB,kd_k = M_{A,k} - M_{B,k}dk​=MA,k​−MB,k​.
  • Гипотезы:
    • H0:μd=0H_0: \mu_d = 0H0​:μd​=0 (Средняя разность метрик равна нулю, т.е. модели не различаются).
    • H1:μd≠0H_1: \mu_d \ne 0H1​:μd​=0 (Средняя разность не равна нулю, т.е. модели различаются).
  • Предположение: Разности dkd_kdk​ распределены примерно нормально (или KKK велико).
  • Статистика Теста (t-статистика):

    t=dˉ−0SEdˉ=dˉsd/Kt = \frac{\bar{d} - 0}{SE_{\bar{d}}} = \frac{\bar{d}}{s_d / \sqrt{K}} t=SEdˉ​dˉ−0​=sd​/K​dˉ​

    • dˉ\bar{d}dˉ: Среднее значение попарных разностей d1,...,dKd_1, ..., d_Kd1​,...,dK​.

      dˉ=1K∑k=1Kdk=1K∑k=1K(MA,k−MB,k) \bar{d} = \frac{1}{K} \sum_{k=1}^{K} d_k = \frac{1}{K} \sum_{k=1}^{K} (M_{A,k} - M_{B,k}) dˉ=K1​k=1∑K​dk​=K1​k=1∑K​(MA,k​−MB,k​)

    • sds_dsd​: Стандартное отклонение попарных разностей d1,...,dKd_1, ..., d_Kd1​,...,dK​.

      sd=1K−1∑k=1K(dk−dˉ)2 s_d = \sqrt{\frac{1}{K-1} \sum_{k=1}^{K} (d_k - \bar{d})^2} sd​=K−11​k=1∑K​(dk​−dˉ)2​

    • SEdˉ=sd/KSE_{\bar{d}} = s_d / \sqrt{K}SEdˉ​=sd​/K​: Стандартная ошибка средней разности.
    • Объяснение: Статистика ttt измеряет, на сколько стандартных ошибок средняя наблюдаемая разность dˉ\bar{d}dˉ отклоняется от нуля (значения, предполагаемого H0H_0H0​). Чем дальше ttt от нуля, тем меньше вероятность получить такую разность случайно, если H0H_0H0​ верна.
  • Принятие Решения (через p-value):
    1. По вычисленному значению ttt и числу степеней свободы df=K−1df = K-1df=K−1 находится p-value.
      • P-value: Вероятность получить значение t-статистики, такое же или более экстремальное, чем наблюдаемое, если нулевая гипотеза H0H_0H0​ верна.
    2. Сравнить p-value с уровнем значимости α\alphaα (обычно 0.05).
      • Если p-value < α\alphaα: Отвергаем H0H_0H0​. Разница между моделями статистически значима.
      • Если p-value ≥α\ge \alpha≥α: Не отвергаем H0H_0H0​. Недостаточно оснований считать разницу значимой.
  • Принятие Решения (через Доверительный Интервал для μd\mu_dμd​):
    1. Строится CI для средней разности μd\mu_dμd​:

      CI for μd=dˉ±tα/2,K−1⋅sdK\text{CI for } \mu_d = \bar{d} \pm t_{\alpha/2, K-1} \cdot \frac{s_d}{\sqrt{K}} CI for μd​=dˉ±tα/2,K−1​⋅K​sd​​

    2. Проверяется, содержит ли CI ноль.
      • Если CI не содержит 0: Отвергаем H0H_0H0​. Разница статистически значима.
      • Если CI содержит 0: Не отвергаем H0H_0H0​.

4. Сравнение Моделей A и B на Одном Тестовом Сете: Бутстрап (Bootstrap)

  • Цель: Оценить стат. значимость разницы метрик Δ=MA−MB\Delta = M_A - M_BΔ=MA​−MB​, имея только один тестовый сет DtestD_{test}Dtest​.
  • Идея: Генерируем множество "псевдо" тестовых сетов DbootD_{boot}Dboot​ путем выборки с возвращением из DtestD_{test}Dtest​. Смотрим на распределение разницы метрик Δboot\Delta_{boot}Δboot​ на этих псевдо-сетах.
  • Алгоритм (Концептуально, без формул вывода):
    1. Повторить RRR раз (например, R=1000R=1000R=1000):
      • Сгенерировать DbootD_{boot}Dboot​ (выборка NNN раз с возвращением из DtestD_{test}Dtest​).
      • Посчитать MA,bootM_{A, boot}MA,boot​ и MB,bootM_{B, boot}MB,boot​ на DbootD_{boot}Dboot​.
      • Записать разницу Δboot=MA,boot−MB,boot\Delta_{boot} = M_{A, boot} - M_{B, boot}Δboot​=MA,boot​−MB,boot​.
    2. Получить распределение из RRR значений Δboot\Delta_{boot}Δboot​.
  • Оценка Значимости (через Перцентильный CI):
    1. Отсортировать Δboot,1,...,Δboot,R\Delta_{boot, 1}, ..., \Delta_{boot, R}Δboot,1​,...,Δboot,R​.
    2. Найти перцентили. Например, для 95% CI:
      • Нижняя граница Δlow=(α/2)\Delta_{low} = (\alpha/2)Δlow​=(α/2)-перцентиль (т.е. 2.5-й перцентиль).
      • Верхняя граница Δhigh=(1−α/2)\Delta_{high} = (1 - \alpha/2)Δhigh​=(1−α/2)-перцентиль (т.е. 97.5-й перцентиль).
    3. 95% CI для разницы метрик: [Δlow,Δhigh][\Delta_{low}, \Delta_{high}][Δlow​,Δhigh​].
    4. Решение: Если интервал не содержит нуля, то наблюдаемая разница Δ\DeltaΔ считается статистически значимой.
  • Оценка Значимости (через p-value - приближенно):
    1. Если исходная разница Δ=MA−MB>0\Delta = M_A - M_B > 0Δ=MA​−MB​>0, то pone−sided=count(Δboot≤0)Rp_{one-sided} = \frac{\text{count}(\Delta_{boot} \le 0)}{R}pone−sided​=Rcount(Δboot​≤0)​.
    2. Если Δ<0\Delta < 0Δ<0, то pone−sided=count(Δboot≥0)Rp_{one-sided} = \frac{\text{count}(\Delta_{boot} \ge 0)}{R}pone−sided​=Rcount(Δboot​≥0)​.
    3. Для двустороннего теста: pvalue≈2⋅pone−sidedp_{value} \approx 2 \cdot p_{one-sided}pvalue​≈2⋅pone−sided​.
    4. Решение: Если p-value < α\alphaα, отвергаем H0H_0H0​.

5. Ключевые Статистические Термины (Резюме)

  • H0H_0H0​ (Нулевая гипотеза): Нет разницы / нет эффекта.
  • H1H_1H1​ (Альтернативная гипотеза): Есть разница / есть эффект.
  • α\alphaα (Уровень значимости): Порог для p-value (обычно 0.05). Вероятность ошибки I рода.
  • P-value: Вероятность наблюдать текущие (или более экстремальные) данные, если H0H_0H0​ верна. (Маленькое p-value   ⟹  \implies⟹ отвергаем H0H_0H0​).
  • CI (Доверительный интервал): Диапазон, который с заданной уверенностью содержит истинное значение параметра. (Если CI для разности не содержит 0   ⟹  \implies⟹ отвергаем H0H_0H0​).
  • SE (Стандартная ошибка): Мера неопределенности / разброса оценки параметра (например, среднего).
